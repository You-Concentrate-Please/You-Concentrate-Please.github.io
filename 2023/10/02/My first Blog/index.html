开始

突然想拥有一个整洁舒适的空间记录自己的一切，学习、工作和生活。我不喜欢杂乱无章，它会影响我记录工作和生活的兴致，在此之前，我的学习和工作空间是一个非常工整有序的文件夹，大多的学习工作文档都是使用vscode加markdown完成，偶尔也会用它来写写生活心得。我很享受所有东西都有组织，但最近总觉得缺乏美感，索性网上找一些教程搭建一个自己的博客空间，尝试在保留我的工作文件夹作为主战场的基础上，将各种分散其中的信息碎片copy并组织到这个空间中，这对于自己也是一件赏心悦目的事。目前博客搭建的只完成了一个毛坯房，总的来说对于像我这样的小白来说还是挺复杂的，包括以后的改进等，还有很多工作要做，干脆把一些关键的步骤和信息先记录下来，然后实时补充更新，以便巩固和分享。

搭建步骤（毛坯房）

我的指导教程来自知乎博主“枫叶”-从零开始搭建博客，基于hexo和Git进行搭建。

绑定Git和Github（工具）

1 需要一个Github账号，并新建一个库，用于后续clone到本地，建库就两步，起名和勾选初始化

2 安装Git，安装一路默认“下一步”（安装路径建议自定义），安装完成后开始菜单打开Git Bash Here，运行命令Git即可验证安装成功

3 绑定Git和Github，目的是方便上传文件（从Git上传到Github），利用SSH登录远程主机完成绑定和文件提交，登录过程选择公钥授权

(1) 生成密钥：新建文件”…\git codelife\text”（随便建，名字随便），用于克隆Github仓库，在这个目录下右键打开Git Bash Here，运行命令ssh-keygen -t rsa，共4次回车生成密钥id_rsa和公钥id_rsa.pub，这两个文件位置Git Bash会显示

(2) 将公钥添加到Github：打开id_rsa.pub，复制，打开Github主页，点击右上角头像下拉菜单点击setting，左侧列表选项点击SSH and GPG keys，点击new SSH key，title可以不填，将之前的公钥粘贴到key，点击add SSH key即可

(3) 完成绑定：在Git Bash中运行ssh -T git@github.com，然后输入yes回车即可

克隆仓库

(1) 打开Github新建的库，在该库的code页，点击绿色的code图标，下拉菜单里clone选项下有HTTPS和SSH链接，复制SSH链接

(2) 进入之前建立的用于克隆仓库的目录，打开Git Bash，运行git clone xxx(刚才复制的SSH链接)

(3) 修改本地仓库并提交至远程仓库（这是个测试步骤）：在克隆到本地的仓库中新建一个文件text.txt,使用命令Git add text.txt将文件添加到临时缓冲区，使用命令Git commit -m，将其提交到本地仓库，第一次提交需要输入输入用户名和邮箱（Github），命令在Git Bash中有提示，在弹出的窗口中选择浏览器同步，将本地仓库修改的内容同步到远程仓库（这一步枫叶的教程中使用命令git push origin main完成）

安装nodejs和Hexo（房子框架）

1 安装nodejs：官网下载安装，一路默认即可，安装完成后cmd执行node -v和npm -v检查，安装完成后设置环境变量，在nodejs文件夹中新建两个文件夹node_cache和node_global，cmd输入命令npm config set prefix "D:\nodejs\node_global"和npm config set cache "D:\nodejs\node_cache"，新建系统变量NODE_PATH，值为D:\nodejs\node_global\node_modules，然后在用户变量里修改npm路径为D:\nodejs\node_global，cmd下执行npm install webpack -g，webpack会下载到D:\nodejs\node_global\node_modules路径

2 安装Hexo：Github新建一个库，命名为yourusername.github.io,在该库的setting界面的pages界面会显示你的库链接，即yourusername.github.io，本地新建...\Blog文件夹，使用Git Bash命令npm install -g hexo-cli安装Hexo，然后使用Hexo init初始化博客，hexo g静态部署，打开...\Blog文件夹下的_config.yml，底部添加

在Git Bash中输入命令npm install hexo-deployer-git --save，安装Git部署插件，然后分别输入

浏览器中打开https://yourusername.github.io即可打开博客

更改域名（住址）

更改域名：在万网-阿里云官网域名页选择并购买域名，需要进行验证和实名等操作，略，认证成功后进行域名解析，点击域名下方的解析，添加两条记录

主机记录 记录类型 记录值
WWW CNAME yourusername.github.io
@ A 你的IPv4地址（ex. 185.199.109.153）

IPv4地址可以在cmd中运行ping yourusername.github.io获得，打开 Blog 文件夹里的 source 文件夹，创建CNAME文件，内容是你买的域名，回到Blog文件夹，Git Bash下运行hexo clean，hexo g && hexo d,回到Github中检查CNAME文件应该已经同步到库中，点击setting，pages页中应该已经自动改成了你的域名，完成
